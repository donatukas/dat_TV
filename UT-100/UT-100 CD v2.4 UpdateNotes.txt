This brief document is for users who have installed UT-100 dongle previously 


Note:
All the applications can only work properly with the new driver, so don't run
 new applications with old driver or old applications with new driver.



How to Update,
 
a. Windows driver
Windows driver: you should goto control panel, device manager, select IT950x, 
click right button, and select "update driver",....
 
b. TS player
Just copy it to your hard disk, and replace the old one. No special install needed.
 
c. PC2TV
Copy it to your hard disk. When the first time use, please run "install.EXE".

d. Media2TS
Copy it to your hard disk. When the first time use, please run "install.EXE".

e. BDAViewer
Copy it to your hard disk. When the first time use, please run "install_filter.bat".

f. DigitalTV
Run Setup.exe to remove the old install, then run Setup again to install the new version.
